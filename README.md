# Facility Maintenance - Управление и обслуживание недвижимости

Сайт для компании по управлению и техническому обслуживанию недвижимости в Уругвае.

## Описание

Сайт ориентирован на владельцев квартир, апартаментов и домов, а также инвесторов и агентств. Предоставляет информацию о пакетах обслуживания и позволяет оставлять заявки через контактную форму.

## Развертывание на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы из этой папки в репозиторий
3. В настройках репозитория найдите раздел "Pages"
4. Включите GitHub Pages, выбрав ветку `main` и папку `/ (root)`
5. Дождитесь деплоя (может занять несколько минут)
6. Сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория`
